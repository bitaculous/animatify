[ANIMATIFY](https://github.com/bitaculous/animatify "Animatify")
================================================================

“All your CSS3 animations are belong to us.”

Installation
------------

1.  Add animatify to your Gemfile:

    ```
    group :assets do
      gem 'animatify', github: 'bitaculous/animatify'
    end
    ```

2.  Run `bundle install`
